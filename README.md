<h2>Pré- requisitos</h2>

1- Instalar node.js

2- Install express:
`npm install express`

3-Instalar hbs:
`npm install hbs`

4- Instalar o sqlite async:
`npm install sqlite-async`

<h2>Como rodar o projeto?</h2>

`npm start`

Entrar na porta: `http://localhost:5500`

Para rodar o banco de dados manualmente: `node src/database/test.js`
